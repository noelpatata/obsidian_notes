- Same Origin Policiy  (SOP) ->  Es una medida de seguridad que restringe la forma en que los documentos o scripts de un origen pueden interactuar con recursos de otro origen, previniendo así posibles ataques. 
- Cross-Origin Resource Sharing (CORS ) -> Es un mecanismo que permite a los supervisores especificar quién puede acceder a los recursos, relajando así las restricciones impuestas por el SOP cuando sea necesario.
