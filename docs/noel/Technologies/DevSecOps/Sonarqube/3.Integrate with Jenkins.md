# Pipeline
Once your pipeline is automatically triggerd thanks to [[configure generic webhook trigger]].
You will add this stage to your Jenkinsfile:
``` yaml
stage('SonarQube Analysis') {

	steps {

		script {

				def scannerHome = tool 'SonarScanner'

				withSonarQubeEnv() {

				sh "${scannerHome}/bin/sonar-scanner"

			}

		}

	}

}
```
# Tool
Before running this pipeline you will need to create the tool you are calling with `tool 'SonarScanner'`. Go to Manage Jenkins > Tools > SonarQube Scanner installations and add one.
![[Pasted image 20260524161819.png]]
# SonarQube server
Then in Manage Jenkins > System > SonarQube servers, add your SonarQube server.
For this step we will need a SonarQube token, we can get that by going to SonarQube > My Account > Security, and then just generate the token.
![[Pasted image 20260524161848.png]]
Now run the pipeline and you will see the scan's overview in the SonarQube UI.
