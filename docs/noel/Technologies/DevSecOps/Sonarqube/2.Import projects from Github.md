# Create Github app
I used [this guide](https://medium.com/@nirashagunawardana9/practical-guide-to-connect-sonarqube-with-your-github-project-sonarqube-part-03-619045f9dd8c) btw
In my case I dont use Github enterprise.
So I navigate to [the github's create app website](https://github.com/settings/apps/new)
I use `sonarqube.downops.win` for the home url, the callback, and the webhook.
I then select the following permissions:
### Repository permissions
- Checks (Read & Write)
- Commit statuses (Read-only)
- Contents (Read-only)
- Pull requests (Read & Write)
### Organization permissions
- Members (Read-only)
- Project (Read-only)
### Account permissions
- Email address (Read-only)
