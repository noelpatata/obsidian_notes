| Symbol | Meaning                                                                     |
| ------ | --------------------------------------------------------------------------- |
| `*`    | You have **modified (dirty)** files (unstaged changes).                     |
| `+`    | You have **staged** changes (i.e., added to the index but not committed).   |
| `%`    | You have **untracked** files in your working directory.                     |
| `#`    | You are on a **detached HEAD** (not currently on a branch).                 |
| `=`    | Your working directory is **clean** (no changes).                           |
| `>`    | Your local branch is **ahead** of the remote (commits to push).             |
| `<`    | Your local branch is **behind** the remote (commits to pull).               |
| `<>`   | Your local branch has **diverged** from the remote (both ahead and behind). |
